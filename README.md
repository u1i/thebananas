# Nano Banana Image Model Evaluation

This repository contains an automated testing and evaluation pipeline designed to objectively compare the zero-shot image-to-image editing capabilities of Google's Gemini generation models. 

## The Target Models

Based on the routing defined in `.env` and `models.csv`:
* **Nano Banana 1**: `gemini-2.5-flash-image`
* **Nano Banana 1 Pro**: `gemini-3-pro-image-preview`
* **Nano Banana 2 (New)**: `gemini-3.1-flash-image-preview`

## Setup & Execution

The testing suite evaluates how well different image models can maintain a reference subject's identity while applying complex environmental effects, lighting situations, and physics defined in text prompts.

1. **Test Definitions**: The 10 rigorous test cases are defined in `tests2.md`.
2. **Reference Image**: The input source image is `u.jpg`. A watermarked version is available at [`labeled_results2/0_Original.jpg`](labeled_results2/0_Original.jpg).
3. **Image Generation**: `run_tests.py` uses the Gemini API's `generate_content` endpoint to run all 10 tests across the 3 target models, outputting raw images to `test_results2/`.
4. **Watermarking**: `add_labels.py` uses ImageMagick to cleanly label the bottom-left of each generated image with its corresponding model name.
5. **Blind Montages**: `create_eval_montages.py` resizes the raw outputs to a uniform height and creates randomized, blindly labeled (1, 2, 3) side-by-side montages of the 3 outputs for each test (saved to `eval_montages2/`), outputting an `answer_key.csv`.
6. **Automated Evaluation**: `auto_eval.py` uses `gemini-3.1-pro-preview` as an automated vision judge to evaluate the blind montages against the original markdown goals, ranking them from 1st to 3rd.

## Evaluation Findings

The full evaluation report generated by `gemini-3.1-pro-preview` is available in [`test_evaluations_report.md`](test_evaluations_report.md).

**The clear overall winner was Nano Banana 1 Pro (`gemini-3-pro-image-preview`)!**

It won an impressive **7 out of 10 tests**, consistently dominating in identity preservation, complex scene interactions (like fogged glasses), and difficult lighting conditions (like intense backlighting or low light inside a tent). 

Here is the breakdown of 1st place finishes:
* 🏆 **Nano Banana 1 Pro:** 7 Wins (Environmental Integration, Background Depth, Material Interaction, Low-Light, Extreme Lighting, Action Framing, Motion Blur)
* 🥈 **Nano Banana 2 (New):** 2 Wins (Extreme Perspective, Physiological Alteration)
* 🥉 **Nano Banana 1:** 1 Win (Multi-Subject Context)

If we assign 3 points for 1st, 2 points for 2nd, and 1 point for 3rd across all ten tests, the final score looks like this:
1. **Nano Banana 1 Pro**: 27 Points
2. **Nano Banana 2 (New)**: 20 Points
3. **Nano Banana 1**: 13 Points

### Key Takeaways
- **Nano Banana 1 Pro** is exceptionally good at maintaining the subject's exact identity (down to the eyewear) while realistically applying physics instructions like steam and motion blur. 
- **Nano Banana 2** is incredibly strong in terms of photorealism and dynamic cinematic angles, but it had a bad habit of unpromptedly removing the subject's glasses or slightly altering facial features to fit a scene. However, it was the *only* model that successfully executed the 'Death Zone' physiological distress test by generating convincingly cracked lips and gaunt skin!
- **Nano Banana 1** struggled the most with complex formatting instructions. It frequently ignored lighting constraints, missed the "selfie" framing, or completely failed physics goals like the fogged glasses.

The heavier "Pro" model still reigns supreme for complex, multi-constraint image-to-image editing tasks requiring high fidelity identity preservation.
